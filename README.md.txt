# Purpose
  This toolbox streamlines combining spatial and tabular data in ArcGIS Pro,
  joining CSVs to GeoJSON Feature Classes, and generating graphs using Matplotlib. 

# Data Accessed
  - CSV files contaning attribute data 
  - GeoJSON files contaning spatial data 
  - Census Data: Income in the Past 12 Months (in 2023, Inflation-Adjusted Dollars)

# How to run
  1. Open the toolbox in ArcGIS Pro. 
  2. Run the tool and select CSV, GeoJSON, output folder, join field, 
     display options, and graph PNG location. 
  3. Click OK to run. Outputs are saved in the selected folder.
